## Business Context

Easy-User là hệ thống quản lý người dùng, đây là dự án phục vụ việc giảng dạy môn ReactJS. Dự án này tập trung vào việc giảng dạy về props, state, cùng với router bằng cách sử dụng `react-router-dom` v7+.

## Tech Stack

- **Framework:** ReactJS
- **Language:** Javascript
- **Styling:** css / inline css
- **State Management:** not yet
- **Package Manager:** NPM
